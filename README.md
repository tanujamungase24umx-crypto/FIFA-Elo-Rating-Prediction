# FIFA Elo Rating Prediction

## Project Overview

This project predicts FIFA World Cup team Elo ratings using Machine Learning.

## Dataset

* FIFA World Cup Elo Ratings Dataset

## Features Used

* Rank
* Wins
* Losses
* Draws
* Goals For
* Goals Against
* Confederation
* Host Status

## Machine Learning Model

* Random Forest Regressor

## Model Performance

* MAE: 10.11
* R² Score: 0.9946

## Sample Prediction

* Predicted Rating: 1556.67
* Actual Rating: 1551

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Google Colab
